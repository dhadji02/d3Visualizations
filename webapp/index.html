This visualization (Line Chart), shows how popularity changes for up to three genres selected in relation to time (years). The time period spans from 1929 to 2022. Popularity is measured by the “Play Count” count of songs.

To create this visualization, the top 100 most popular songs were used. This visualization (Histogram), represents the frequency of each music key in which each of the top 100 songs has been written and for each genre, individually. The numbers inside the bars of the Histogram are expressed in percentages.
The first visualization displayed (Line Chart), shows how popularity changes in relation to the release month of songs. The second visualization which could either be a line chart or a heatmap shows how popularity changes in relation to the release weekday of songs. The “Play Count” of each song is expressed as a score which is represented as popularity. When the user clicks a tick on the axis, that represents the month in the first visualization, the second one adapts to this selection. In order to calculate the popularity, the charts take into account data from 1929-2022.
